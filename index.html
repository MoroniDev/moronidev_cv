Proyectos destacados

Automatización de recibos de nómina

Aplicación en C# que convierte carpetas completas con archivos XML en recibos de nómina en PDF usando una plantilla definida, reduciendo horas de trabajo manual.

Fusión de XML contables

Herramienta que unifica información contable en un solo archivo estructurado, listo para su uso dentro de sistemas administrativos.

Pólizas y cuentas contables CONTPAQi

Programa en C# que lee un archivo Excel, crea cuentas contables y genera pólizas dentro de CONTPAQi utilizando su SDK, automatizando la captura manual.

Consumo automático de API de pagos

Integración con la API de Clip que obtiene reportes de pagos y los guarda en la base de datos de CONTPAQi Comercial, actualizando automáticamente el estado de las órdenes.
